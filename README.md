# [ZP] Packs Shop

A time-limited shop system for Zombie Plague X that allows players to exchange Points for Packs.
Players can purchase different Packs packages using their Points. Each package can only be purchased once per map, and the shop is available only during a specific time period.

## Plugin Information

  - **Name**: [ZP] Packs Shop
  - **Version**: 1.0
  - **Author**: DadoDz
  - **Game**: Counter-Strike 1.6
  - **Mod**: Zombie Plague

## Requirements

- AMX Mod X **1.9+**
- Zombie Plague Mod

## Installation
1. Place 'zp_shop_menu.sma' in: addons/amxmodx/scripting/
2. Compile it with your AMXX compiler.
3. Place the compiled .amxx file in: addons/amxmodx/plugins/
4. Add this line to your plugins.ini: zp_shop_menu.amxx
5. Restart your server.

## Commands

| Command | Description |
|-------|-------------|
| `/packs` | Opens the Packs Shop |
| `packs` | Opens the Packs Shop |
| `/shop` | Opens the Packs Shop |
| `shop` | Opens the Packs Shop |

## Notice
> This plugin was created specifically for Zombie Plague X and uses the Points System and Packs System from Zombie Plague X.
> The shop can be adapted to work with other Zombie Plague versions or other game mods by modifying the required systems and natives.
